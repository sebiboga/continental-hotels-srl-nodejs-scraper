Continental Hotels Scraper

Scraper pentru platforma peviitor.ro.

Structură

  • index.js — orchestrator principal
  • company.js — validare companie (ANAF + Peviitor + SOLR)
  • solr.js — operațiuni SOLR
  • src/anaf.js — bibliotecă ANAF

Rulare

npm run scrape        # rulează scraperul
npm test             # rulează teste unitare

Variabile de mediu

SOLR_AUTH — credentiale SOLR (opțional pentru teste)
